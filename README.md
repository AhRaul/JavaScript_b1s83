# JavaScript_b1s83
Версия 1 игры, в морской бой.
Здесь задано одномерное поле, со значениями от 0 до 6 (включительно).
Т.к. это 1, упрощенная, подготовительнаая версия программы, тут не генерируется случайное положение корабля.
Корабль, по умолчанию, всегда находится на заранее заданных клетках.
3,4,5.
Игрок вводит во всплывающем окне браузера точкУ, в которую хочет выстрелить.
Игра считает выстрелы.
В конце, когда игрок попадет во все точки, в которых находится корабль, игра считается пройденой.
Вылезет окно, с информацией, о количестве выстрелов и вероятности попаданий.
